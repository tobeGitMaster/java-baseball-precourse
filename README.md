# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

# 기능 목록
1. 3자리 랜덤 숫자 생성기
2. 게임 결과(볼, 스트라이크 등) 객체 생성
3. 게임 플레이 관련 서비스 생성
4. 게임 플레이 관련 컨트롤러 생성
5. 화면(뷰) 입력 구현
6. 입력 유효성 체크 기능 구현
